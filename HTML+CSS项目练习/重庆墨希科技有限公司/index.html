重庆墨希科技有限公司是上海南江集团与中国科学院重庆绿色智能技术研究院共同出资 成立的一家高新技术企业，公司成立于2013年3月，首期投资2.67亿元人民币。 墨希科技是高新技术企业，致力于石墨烯薄膜材料规模化生产、 石墨烯薄膜材料规模化制备成套装备及系统解决方案研发、石墨烯应用产品开发。 2016年12月成为华丽家族股份有限公司（证券代码600503）旗下科技板块的高科技公司。 2017年公司被认定为高新技术企业。
